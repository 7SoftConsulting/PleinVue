# PHP-PV

## Introduction

**PHP-PV** signifie **PHP-Pleine Vue**. C'est un ensemble de librairies PHP pour réaliser des applications robustes.

## Caractéristiques

- Orientée Objet
- Ecrit en Francais
- Convention de Nommage : PascalCase
- Inspiré de la VCL de Microsoft .NET 4.0
